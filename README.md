# ML-KEM

This is a clean room implementation of Module-Lattice-Based Key-Encapsulation Mechanism described in [ML-KEM](https://nvlpubs.nist.gov/nistpubs/fips/nist.fips.203.pdf) paper.

I wanted to understand and implement algorithms without looking at other implementations.
